# TECH DE BAIRRO

O Tech de Bairro é um programa de inclusão digital que leva capacitação tecnológica diretamente para dentro das comunidades e escolas públicas.
Com metodologia prática, acessível e acolhedora, o projeto conecta jovens e adultos às competências do futuro — para que ninguém fique para trás na revolução digital.

O Tech de Bairro opera em quatro frentes principais:

- Website informativo sobre o projeto, onde os interessados podem se inscrever no programa;
- Envio mensal de materiais educativos para os inscritos;
- Redistribuição de equipamentos tecnológicos doados;
- Trabalho em rede com parceiros sociais e institucionais.
